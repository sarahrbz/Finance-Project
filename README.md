# 💰 Sistema Financeiro - Angular

Aplicação web de controle financeiro desenvolvida com Angular e Bootstrap.

Este projeto é uma reconstrução de um sistema desenvolvido em aula, agora utilizando Angular e integração com Supabase para gerenciamento de dados.

## 🚀 Tecnologias

- Angular
- Bootstrap
- TypeScript
- Supabase

## 📚 Projeto acadêmico

Desenvolvido para fins de estudo no curso de Análise e Desenvolvimento de Sistemas.
